# Kopi KHA Landing Page

Next.js landing page for Kopi KHA, ready for Vercel deployment.

## Development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy to Vercel

1. Push this folder to a GitHub repository.
2. Import the repository in Vercel.
3. Use the default Next.js settings.
4. Deploy.

All incomplete business data is intentionally kept as bracketed placeholders.
